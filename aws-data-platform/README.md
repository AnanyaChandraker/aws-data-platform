# AWS Data Platform - Example Repository
This repository demonstrates a reference implementation for an enterprise AWS data platform using:
- Kinesis (real-time ingestion)
- AWS Lambda (processing)
- S3 (raw/processed/curated storage)
- EMR (batch PySpark processing)
- Glue (catalog + serverless ETL)
- MWAA (Airflow orchestration)
- Simple Data Quality (DQ) framework
